zookeeperTest
^^^^^^^^^^^^^
::

    这是一个zookeeper的python测试程序

目录结构
-------
+ bin
    |执行程序
+ conf
    |配置文件
+ src
    - app
        |应用逻辑
    - utility
        |通用工具
    - other
        |其他包
    - main.py
        |主程序
+ setup.py
    |安装部署
+ README.rst
    |说明文档

执行
----
::
    |run.sh start ../conf/conf.cnf
    启动程序
