zookeeperTest
^^^^^^^^^^^^^
::

    这是一个zookeeper的python测试程序，用于测试主从模式的实现

目录结构
-------
+ bin
    执行程序
+ conf
    配置文件
+ log
    日志文件目录
+ src
    - utility
        通用工具
    - zk
        zookeeper相关
    - zookeeperTest.py
        主程序入口
+ setup.py
    安装部署
+ README.rst
    说明文档

执行
----
::

    run.sh start ../conf/conf.cnf
    启动程序

版本
----
::

    v1.0 初始代码
